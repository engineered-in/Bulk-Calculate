What is Bulk Calculate?

Bulk Calculate is a free and open-source Excel utility designed to streamline repetitive, template-based engineering calculations.

By centralizing inputs and outputs into a single summary table, it helps engineers focus on their work instead of managing countless files.

Ideal for civil, mechanical, chemical, and electrical engineers, Bulk Calculate makes engineering projects more data-driven and efficient.

Easily map data in your templates: Simply point, click, and let the wizard handle the rest!

Key Features

Template-based Calculations

Use any of your Excel calculation file as a template, and perform bulk operations efficiently.

Map Data Wizard

Easily map calculation input and output cells in your template file using our intuitive Map Data Wizard.

Summary Table

Get a comprehensive overview of all calculation inputs and outputs in a single summary table.

Bulk Export and Import

Export individual spreadsheets or PDFs, and import data into Bulk Calculate—all within Excel.
